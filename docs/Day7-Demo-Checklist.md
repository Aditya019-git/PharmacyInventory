# Day 7 Demo Checklist

## 1. Start

- Login as Admin.
- Show dashboard stats.

## 2. Core Flows

- Open Medicines and add one medicine.
- Open Suppliers and add one supplier.
- Open Inventory and add one batch with expiry.
- Show stock adjustment in/out.

## 3. Expiry

- Open Expiry Alerts and change day filter.
- Highlight near-expiry and expired sections.

## 4. Billing

- Open Billing and add item to cart.
- Checkout with payment mode and optional discount/tax.
- Open generated invoice and print preview.

## 5. Reports

- Open Reports.
- Filter by date and payment mode.
- Show KPI totals + low stock + expiry risk.
- Open invoice link from sales report.

## 6. Role Control

- Open Pharmacists module (admin only).
- Create a new pharmacist user.
- Logout admin and login as that pharmacist.
- Confirm pharmacist cannot access `/pharmacists`.

## 7. Wrap-up

- Mention audit logs for login/logout/pharmacist creation.
- Mention stack: Java, JSP/Servlet, JDBC, MySQL, Tomcat.
